# Design-Patterns-Project
This a project, for COS 214, where a group of people will model a Formula one team using at least 10 design patterns. 

The following pattersn were used:
    
      1. Composite
      2. Decorator
      3. Prototype
      4. Factory
      5. Builder
      6. Memento
      7. Iterator
      8. State
      9. Observer
     10. Mediator
     11. Strategy
