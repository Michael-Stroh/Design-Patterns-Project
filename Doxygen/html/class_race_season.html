Result* RaceSeason::runSeason ( )

Runs the entire race season.

Will iterate through the collection of GrandPrixs, running each individually calling the GrandPrix.runGrandPrix() method. This will return a result for each GrandPrix, which is added to the SeasonResult (thus aggregating each GrandPrixs result into a final RaceSeasonResult).

Returns
A RaceSeasonResult object, that can be used to access the results of the entire RaceSeason.
