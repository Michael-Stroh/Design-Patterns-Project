virtual CarPart* Simulation::simulate ( CarPart *  ,
float  variance[],
float  max[]

pure virtual

Clones the input car part, then adjusts it stats according to the values in the variance array while being sure that none exceed the corresponding values in the max array. Returns the Better car part (without deleting anything, deletion takes place in the Engineering Department that calls this function.

Parameters
thecarPart to be worked on
anarray of variances that model how much the acceleration, handling and speed can change respectively
anarray of the max values for the acceleration, handling and speed respectively
Returns
The better CarPart out of the cloned part and the new part.

Implemented in WindTunnel, SpeedSimulation, HandlingSimulation, and AccelerationSimulation.
